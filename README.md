# Automated Script Tools on Linux
This repo contains a series of automated bash scripts that uses a variety of libraries and tools made for Linux. <br/>
Some of them contain utilities like nmcli, aircrack, etc. So to completely use this repo it is a requirement to install the respective tools. <br/>
Inside every directory there may be 1 or 2 bash files that contain the neccesary commands and a README.md which explains how they work and how to use them. <br/>
